# Face Login Prototype

## Goal

Build a simple face login prototype using React and FastAPI.

The project is intended for learning and demonstration purposes.

## Features

### Register

* Enter email
* Capture face image
* Store user data

### Login

* Enter email
* Capture face image
* Verify face

### Dashboard

* Show welcome message after successful login

## Success Criteria

A user can:

1. Register.
2. Login using face verification.
3. Access the dashboard.

## Out of Scope

* OTP
* JWT
* Admin panel
* Cloud deployment
* Liveness detection
* Mobile application
